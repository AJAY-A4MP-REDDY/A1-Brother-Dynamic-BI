# 📊 A1 Brother Dynamic Power BI

A dynamic, browser-based BI (Business Intelligence) dashboard built with **Flask** and **Plotly**.  
Upload any CSV file and instantly visualize it with **10 chart types**.

---

## 🚀 Features
- 📁 Upload CSV, Image, or any file
- 📊 10 Chart Types (Bar, Line, Pie, etc.)
- 📱 Mobile-friendly responsive design

## 🛠️ Installation & Run

```bash
# Clone the repo
git clone [https://github.com/AJAY-A4MP-REDDY/A1-Brother-Dynamic-BI.git](https://github.com/AJAY-A4MP-REDDY/A1-Brother-Dynamic-BI.git)
cd A1-Brother-Dynamic-BI

# Install dependencies
pip install -r requirements.txt
